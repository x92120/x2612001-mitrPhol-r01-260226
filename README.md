# Gemini 3.1 Explorer

This project is set up to explore the capabilities of Google's Gemini 3.1 Pro model, including its reasoning (thinking) levels and multimodel capabilities.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Add your API key to a `.env` file:
   ```
   GOOGLE_API_KEY=your_key_here
   ```

4. Run the explorer:
   ```bash
   python explorer.py
   ```
