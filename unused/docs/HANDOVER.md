    - **Fix:** Added a `/pre-batches/` alias in `router_production.py`.

    - The server was crashing during login due to a version conflict between `passlib` and `bcrypt 5.0.0` (Error: `password cannot be longer than 72 bytes`).
    - **Fix:** Downgraded `bcrypt` to `4.0.1` in the virtual environment.
